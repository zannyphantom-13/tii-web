❓ NEED HELP?

    Problem: Port 3000 already in use
    Solution: $env:PORT=5000; python app.py

    Problem: ModuleNotFoundError
    Solution: pip install -r requirements.txt

    Problem: Cannot activate venv
    Solution: python -m venv venv (recreate it)

    Problem: API returns 404
    Solution: Check Python service is running, check browser console

    More help in: README_SETUP.md or QUICKSTART.md
